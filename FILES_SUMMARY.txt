【数据库模型】

User 模型:
- id: String (主键)
- email: String (唯一)
- name: String
- password: String
- createdAt: DateTime
- updatedAt: DateTime

Category 模型:
- id: String (主键)
- name: String (唯一)
- slug: String (唯一)
- description: String (可选)
- icon: String (可选)
- createdAt: DateTime
- updatedAt: DateTime
- products: Product[] (关系)

Product 模型:
- id: String (主键)
- name: String
- slug: String (唯一)
- description: String (可选)
- price: Float
- image: String (可选)
- category: Category (关系)
- categoryId: String (外键)
- featured: Boolean
- rating: Float
- reviews: Int
- createdAt: DateTime
- updatedAt: DateTime
